# Evangadi Forum

A full-stack question-and-answer forum where users can sign up, ask questions, answer, and interact in real-time.

##  Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MySQL (via XAMPP)
- Authentication: JWT (JSON Web Token)
- Additional: React Router, Bootstrap, Axios

##  Features
- User registration and login with JWT authentication
- Post, edit, and answer questions
- Search and filter questions
- Responsive design for desktop and mobile
- Admin panel to manage users

##  Setup
```bash
# Clone the repository
git clone https://github.com/medoshaweb/evangadi-forum.git

# Backend
cd evangadi-backend
npm install
nodemon server.js

# Frontend
cd evangadi-frontend
npm install
npm run dev
